# 满满漫画工坊

这是一个可直接部署到 GitHub Pages 的静态页面。

## 使用方式

1. 打开页面
2. 填入育儿日常文案
3. 上传 AI 生成的空白漫画底图，或使用内置示例底图
4. 点击“生成漫画方案”
5. 切换查看“空白分镜草图”“带字分镜草图”“对话文案”“AI 生图提示词”
6. 下载 PNG 或复制小红书发布内容
7. 点击“打开小红书发布页”完成发布

## GitHub Pages 部署

在仓库设置里启用 Pages：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

## 说明

当前版本是纯前端静态页面。推荐流程是先用“AI 生图提示词”生成空白气泡漫画，再把空白漫画上传回页面，页面负责叠加中文对白并导出带字 PNG。

如果后续要接真实 AI 生图，建议新增一个后端服务，由后端保存 API Key 并提供漫画生成接口。
